Who We Are

Kilowatt is a tech start-up based in the Northern Virginia/DC area. We are focused on system architecture, data science and web development.

With a strong dependency on data, we are able to automate any process or task while keeping the organic integrity of a system intact.

Plan

Most of our ideas start on paper during the planning proccess. With us no idea is too big. During our planning process, we focus on user experience and data flow.

Design

We use markup technology like FIGMA and Adobe XD to properly plan out a systems methodology. This helps us minimize development time. Before we build, all designs must go through a peer review.

Build

Once we have designed our markup, we began prototyping the idea within a developer environment. This gives us a better overhead view on what needs to be approved before being pushed to production. During this process all code is put through peer reviews and tested for bugs, data leaks and initial glitches.

Launch

Once our project has gone through the planning, designing, and building process it is now time to create a git repository to control versioning of the project being engineered. This ensures that when something goes wrong, we can invert back to a stable or working version of the project. Once we have control over versioning we simply launch the project via cloud or through an application packaging system.
